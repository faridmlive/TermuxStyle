# NO EXTERNAL DEPENDENCIES REQUIRED
# ============================================================================
#
# This project uses only Python standard library modules:
#   - os (for file operations and system calls)
#   - sys (for system-specific parameters)
#   - random (for random quote selection)
#   - time (for sleep/animations)
#   - shutil (for file operations)
#
# Python Version Required: >= 3.6
#
# ============================================================================
# INSTALLATION
# ============================================================================
#
# Python 3.6 or higher is required
#
# Install Python on Termux:
#   pkg install python -y
#
# Install Python on Linux:
#   sudo apt install python3 -y  (Debian/Ubuntu)
#   sudo dnf install python3 -y  (Fedora)
#   sudo pacman -S python         (Arch)
#
# ============================================================================
# USAGE
# ============================================================================
#
# Simply run the generator script:
#   python TermuxStyle.py
#
# Or on Linux:
#   python3 Terminal_Generator.py
#
# ============================================================================
# OPTIONAL DEPENDENCIES (Not required, but useful)
# ============================================================================
#
# For development or contribution:
# - git (for cloning and version control)
#
# Install git on Termux:
#   pkg install git -y
#
# Install git on Linux:
#   sudo apt install git -y
#

# NOTES
# ============================================================================
#
# - This script is self-contained and portable
# - No pip packages need to be installed
# - Works on Python 3.6+
